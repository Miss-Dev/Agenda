# Agenda de contatos

Esta é uma agenda simples de contatos utilizando frameork Django e banco de dados com a área administrativa do Django. 

### Rodando a aplicação

Recomendo utilizar ambiente virtual python, para rodar a aplicação. Após isso basta instalar o Django com o comando => *py -m pip install Django*

Após isso executar o comando => *python manage.py runserver*
